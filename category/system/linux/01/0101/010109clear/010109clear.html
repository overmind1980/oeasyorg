特殊情况

有些文件问不出具体位置,比如

  • cd
  • pwd
  • 说他们是shell built-in command
这些命令在哪?🤔
  • 这些命令,是在这个shell里面的
  • shell启动的时候,就在内存中了
  • 所以叫shell built-in command
  • 没有具体的硬盘对应的命令文件
  • shell啥时候启动的?
  • 终端terminal启动时候,把硬盘上的shell代码(如zsh),加载到内存里
  • 如果要找相应的源代码,就去shell的源代码里面找
