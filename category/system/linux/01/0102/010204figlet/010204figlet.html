figlet 细节

  • 对齐方式：
    • -l 左
    • -c 中
    • -r 右
  • 改变字体：
    • -f fontfile
    • figlet 字体文件 flf 在哪呢？🤔
#使用 find 命令在/(root) 下查找所有 flf 文件
sudo find / -name "*.flf"
