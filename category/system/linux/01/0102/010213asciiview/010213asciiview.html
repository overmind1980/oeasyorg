下载aview🅰️
  • 字符画软件包aview、imagemagick
sudo apt install aview
sudo apt install imagemagick
  • aview只接受专门的格式
  • 使用asciiview可以转化格式
  • bmp格式更容易被接受⚠️
asciiview panda.jpg
存储字符画
  • 通过查询帮助📕,发现字符画可以保存
  • 可以把字符画保存为文本、网页等形式
  • 快捷键是s,然后一路下一步
  • 最后存在工作文件夹pwd
结合cowsay🐄
# 在确认cowsay已经安装的情况下
cd /usr/share/cowsay/cows
# 复制一个cow文件
sudo cp apt.cow oeasy.cow
# 是用编辑器编辑
sudo gedit oeasy.cow ~/Code/ooo.txt
