oneko🐱

我们可以下载并运行这个小应用
sudo apt install oneko
oneko
  • 然后就会出现一只小猫🐱
  • 小猫会跟着你的鼠标走
  • 走到你的鼠标🐭附近她还会撒娇
  • 可以把桌宠换成狗🐶、木之本樱?大道寺知世?
  • 可以控制时间间隔、移动速度
  • -towindow可以让她指定停靠在某个窗口上
  • 可以设置前景背景色
  • 我们再来看另一个桌宠
