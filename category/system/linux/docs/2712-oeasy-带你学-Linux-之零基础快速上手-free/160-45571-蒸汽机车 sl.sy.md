## 动力机车 sl

我们可以这样写：

```shell
sudo apt install sl
sl
sl -l
sl -F
```

**一回车，我们竟然看到一列火车隆隆驶过！** 😱

**赶快查询一下手册,我们看到：**

`SL` 代表 `Steam Locomotive`，这个命令和 ls 很像，这个命令就是在 ls 打错的情况下提提神：😓 

- `-l` 选项:会出现一个小火车
- `-F` 选项:会让火车越飞越高，要注意是大写F
- `-e` 选项:允许中途被 <kbd>ctrl</kbd>+<kbd>c</kbd> 打断
