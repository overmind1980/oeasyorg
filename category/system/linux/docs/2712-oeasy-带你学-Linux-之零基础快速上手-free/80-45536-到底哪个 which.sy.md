### 我们来玩转 which 吧 🤗

**各种命令都来当 which 的参数，比如：**

- which pwd
- which uname
- which whatis
- which whereis

现在我们有了三个灵魂问题了： ✊

- whatis 你是谁
- whereis 你在哪
- which 到底在哪

通过这三个命令我们可以知道，任何命令的作用、位置，我们给这三个问题起名叫**灵魂三问**！👊

### 灵魂三问

接下来我们来对 `cat` 命令，试试这个灵魂三问：

```shell
whatis cat
whereis cat
which cat
```
