## 黑客帝国 Matrix 📟

```shell
apt search matrix
apt show cmatrix
sudo apt install cmatrix
```

我们发现一个叫做 cmatrix 的软件包，下载需要 2M，好像可以玩，走起来！

```shell
cmatrix
```

**那么我们可以看看手册里的选项细节吗？**

### 矩阵细节 📟

- `-b`: 部分加粗头部变色 bold
- `-B`: 全部加粗头部变色 Bold
- `-o`: 老式的 old school
- `-s`: 屏保模式 Screensaver
- `-u`: 我们可以设置中间 0-9 的间隔时间 ⏲
  - 0 为无间隔非常快
  - 9 为比较慢
- `-C` 后面加上想要的颜色 (Color) 模式
  - 目前支持红绿蓝青品黄黑白
  
**让我们来“举个🌰 栗子”：**

```shell
cmatrix -b
cmatrix -u 9
cmatrix -C blue
```
