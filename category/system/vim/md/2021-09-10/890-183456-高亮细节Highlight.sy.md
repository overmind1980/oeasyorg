# 高亮细节 highight

## 回忆
- 这个自动命令 `autocmd` 还是很方便的
- 打开时、保存时就会有自动执行的操作
- 自动命令有这么几大元素
	- `{event}` 触发事件
	- `{pattern}` 文件模式
	- `{cmd}` 具体执行命令
	- `{augroup}` 命令组
- 自动命令可以新建、删除、列表、查询
- 还可以强制执行
- 有这个我们可以
    - 针对每种不同的文件的类型
    - 定义相应的触发事件
    - 然后执行各种各样的命令
    - 方便操作
- 不过关于文件类型的高亮显示还是没有讲的特别清楚
    - 为什么`public`在`java`文件里面就可以改变颜色呢？？🤔

### 设置高亮

- `:colorscheme murphy`
- `:hi String ctermfg=brown`
	- hi 是highlight
	- String是需要定义颜色的组名称
	- cterm是color terminal foreground
	- brown 是棕色

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20210802-1627909966231)
